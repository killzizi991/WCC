ЗАПРОС НА РАЗРАБОТКУ. ФОРМАТ ОТВЕТА ЖЕСТКО РЕГЛАМЕНТИРОВАН.

ЗАДАНИЕ:





ТРЕБОВАНИЯ К ФОРМАТУ ОТВЕТА:
Выдавать только код файлов изменённых под задачу
Запрещено любое текстовое пояснение, предваряющее код.
Запрещено выводить код в формате обычного текста.
Ответ должен состоять ТОЛЬКО из последовательности блоков:
Строка с указанием файла: // FILE: полный/путь/к/файлу.расширение
Немедленно после нее — блок кода, обрамленный тройными бэктиками с указанием именем файла, содержащий ПОЛНЫЙ код этого файла от первой до последней строки.
Запрещены любые сокращения, комментарии типа // ..., пропуски кода.
Если все файлы не помещаются в один ответ, вывод должен быть прерван строкой, со списком оставшихся файлов для изменения под задачу ( под **** должно быть название файлов которые ещё нужно изменить под задачу): [ERROR: OUTPUT TRUNCATED. TYPE 'CONTINUE XZ'] Оставшиеся файлы -****

КРИТИЧЕСКИ ВАЖНО:
Любое отклонение от данного формата является невыполнением задания. Весь выдаваемый код предназначен для прямого копирования в файлы проекта.
